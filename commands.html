COMANDOS

Interações

Utilize a tecla de espaço para interagir em diversas partes do jogo, como iniciar um minijogo ou abrir uma tarefa.

Pause

Utilize a tecla "P" ou "ESC" para pausar o jogo. Os mesmos também podem ser utilizados para retomar o jogo.

Clique

Utilize o clique esquerdo do mouse para selecionar itens ou o mantenha segurado para arrastar objetos.

Movimen-
tação

Utilize a tecla "A" para movimentar-se à esquerda, tecla "D" à direita, tecla "W" para cima, e tecla "S" para baixo.
